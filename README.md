Jarvis — Lightweight AI Voice Assistant

Overview

Jarvis is a lightweight AI voice assistant built in Python. 
It can understand natural language commands, execute multi-step workflows,
launch applications, and run PowerShell commands. Designed for minimal resource usage, 
Jarvis runs efficiently in the background and can operate on low-spec systems.



Features

* Conversational AI with natural language understanding
* Lightweight backend that runs with minimal CPU and memory usage
* Application launching and automation based on user commands
* PowerShell integration for executing system-level tasks
* Interruptible voice output — allows users to stop or pause speech
* Background noise management for accurate voice recognition



Technologies Used

* Python
* SpeechRecognition / PyAudio
* OpenAI
* Threading and audio management modules
